# Documentation details

## Live preview

<div style={{ display: 'flex', alignItems: 'center' }}>
    Sometimes you'll see <Icon style={{ marginBottom: 6 }}>whatshot</Icon> icon.
</div>
This means that this code can be edited and you will immediately see the result.

You can provide just jsx, function component or class component:

```jsx live
<div>it works!</div>
```

```jsx live
() =>
    <div>Function component!</div>
```

```jsx live
class ClassComponent extends React.Component {
    render() {
        return (
            <div>Class component</div>
        );
    }
}
```

### Errors

If an exception will thrown, you will immediately see it:

```jsx live
<div>
    Trying to access not existing variable {lol}
</div>
```

### Scope

In live editor you can get React and any Morfix library exported member, like:

- Morfix
- Field
- RealtimeField
- ObjectField
- and others

### Limitations

Live editor not supports imports/requires

### Try it!

```jsx live
<Morfix inititalValues={{ name: "Edit me!" }}>
    {({ values }) =>
        <React.Fragment>
            {values && values.name}
            <RealtimeField name="name" />
        </React.Fragment>
    }
</Morfix>
```

### More information

We use [react-live](https://github.com/FormidableLabs/react-live) libaray for live editor.
